全站 UI 視覺總覽

把 91 頁逐頁盤點到的每個 UI 元素，真的畫出來給你看：左邊綠底是「✅ 標準做法（共用組件 + morandi 色票）」、右邊紅底是「🔴 異常做法（手刻 / Tailwind 預設色 / 失效 token）」。每個樣本旁邊都標「出現在哪些頁」。一眼看出哪裡長得不一樣、該收編成哪一套。
✅ 標準（走共用組件 + token） 🟡 半對齊（組件對、樣式手刻） 🔴 異常（手刻 / 預設色 / 失效）
資料來源：UI-元素出處對照.md（91 頁盤點）｜真實色票複刻自 src/styles/tokens.css morandi 預設主題｜圖示自 lucide-react v0.544 抽取｜2026-05-26
大前提：全站不是一套 UI、是三套設計系統並存。 ① 桌面 ERP（morandi 金色、本總覽主範圍）　② 行動版 /app（藍色深色、手刻 CSS、0 共用組件、另一個世界）　③ 對客頁（提案 / 付款 / canvas，多套獨立 CIS，William 拍板的特例、暫不強制納管）。下面的「異常」多數指 ERP 內部該對齊卻沒對齊的部分。
元素類型現況幾套做法→ 建議統一成
1按鈕 Button共用 5 variant + ActionCell；另有手套骨架、純手刻、手寫色蓋 variant、認證頁 3 套、行動版另一套主/次走 <Button> variant、操作欄走 <ActionCell>

6頁籤 Tabs框架 tabs + shadcn Tabs（2 套清楚）；另有手刻 button tab、dead-ish _TourTabs路由級走框架 tabs、內容級走 shadcn Tabs

8分頁 PaginationEnhancedTable 內建（固定 15 筆）；/finance 自刻分頁列；visas/shared-data 全量無分頁一律 serverPagination 固定 15 筆
