### A1. ESLint ratchet baseline（today snapshot）

`.eslint-suppressions.json` 真實統計：

| Rule                                | 違規數  | 違規檔數     |
| ----------------------------------- | ------- | ------------ |
| `venturo/no-direct-supabase-writes` | 127     | 53           |
| `venturo/no-direct-useswr-in-pages` | 18      | 18           |
| **合計**                            | **145** | **68 files** |

機制（已在 `package.json`）：

- `npm run lint` 用 `.eslint-suppressions.json` 過濾 baseline、**新增違反直接擋**

- `npm run lint:swr-prune` 自動移除已修好的條目（修完後自動清）

✅ **ratchet 機制已落地**、不需要新做。
