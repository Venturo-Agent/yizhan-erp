# 公司設定頁 — 重構待辦

> 記錄：2026-05-28（Logan）｜ William 拍板

## 待辦：財務相關設定搬到「財務設定」

把以下四塊從「**公司設定**」頁搬到「**財務設定**」（`/finance/settings`）：

1. **結帳設定**（預設出帳日期 `default_billing_day_of_week`）
2. **匯款手續費**（模式 `transfer_fee_mode` 平均分配 / 統一收付 + 統一金額 `transfer_fee_unified_amount` + 溢額入帳 `transfer_fee_overflow_account_id`）
3. **結帳稅率**（`profit_tax_rate`，存在 `workspace_bonus_defaults` PROFIT_TAX row）
4. **銀行資訊**（`bank_code` / `bank_name` / `bank_branch` / `bank_account` / `bank_account_name`）

### 為什麼

這些是**財務相關設定**、不該放在「公司基本設定」。
公司設定頁最終只留：公司識別 + 聯絡資訊 + Logo + 公司印章。

### 搬遷時要動（給未來做的人）

- `CompanyInfoCard.tsx`：移除「結帳設定」區（目前 ~289 行起）的這些欄位 UI
- **財務設定頁**（`/finance/settings`）：新增這些欄位 + 存檔邏輯
- `settings/company/page.tsx`：form mapping / 存檔 payload（這些欄位的讀寫搬走）
- `settings/company/types.ts`：`CompanyFormData` 的這些欄位搬到財務設定的 form type
- DB：欄位本身在 `workspaces` 表（bank_* / default_billing_day_of_week / transfer_fee_*）、稅率在 `workspace_bonus_defaults`——**搬 UI 不一定搬 DB 欄位**，看財務設定頁怎麼接（保留 DB 欄位、只搬 UI 最安全）

### 注意（紅線）

- 這是「移動既有功能」、動手前先探查財務設定頁現況 + 這些欄位的存檔路徑（憲法第 9 條）
- 結帳稅率存在 `workspace_bonus_defaults`（不是 `workspaces`）、搬時別搞混
