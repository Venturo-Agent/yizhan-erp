## 怎麼用這個資料夾

- 每個頁面 / 功能一張 `.md` 卡。
- 每張卡分兩段：
  - 🔧 **工程脈絡**（給 AI / 工程師）：路由、定位、功能、權限、資料、關聯、設計沿革。
  - 📖 **使用教學**（給使用者）：白話操作步驟、截圖位置、注意事項。
- William 慢慢截圖、慢慢做；Logan 一塊塊探查、寫卡。不趕、增量累積。
