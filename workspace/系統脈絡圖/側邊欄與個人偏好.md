### 路由 / 入口
- 全站皆可見，常駐於 `(main)` layout 左側。
- 組件：`src/components/layout/sidebar.tsx`
- 設定 SSOT：`src/components/layout/sidebar-config.ts`

### 功能清單
1. **主選單**：17 個模組（見 README 網站地圖），4 個有子選單（財務 / 會計 / 人資 / 資料管理）。
2. **權限過濾**：項目依 `workspace_features`（租戶有沒有開）+ `role_capabilities`（員工能不能讀）自動顯示 / 隱藏。
3. **展開 / 收合**：點擊固定展開、滑鼠懸停臨時展開。**狀態不持久化**（重整回收起）。
4. **底部使用者區**：頭像圖示、名字、個人偏好齒輪、登出。

### 權限 / 可見性
- 每項由 `requiredPermission` + workspace feature 雙重過濾（`sidebar.tsx` line 147-174）。
- 使用者**不能**自訂順序 / 釘選 / 隱藏——純權限驅動。順序固定於 `SIDEBAR_ORDER`。

## 📖 使用教學（給使用者）

### 這是什麼
登入後左邊那條就是**主選單**，整套系統的功能都從這裡進去。

### 主選單怎麼用
- 滑鼠移上去會展開、看到每個功能的名字；點圖示可以固定展開。
- 名字右邊有「›」箭頭的（財務系統、會計系統、人資管理、資料管理），點下去會展開**子選單**。
- 💡 **你看到的選單，是依你的權限顯示的**——同事看到的可能跟你不一樣，這是正常的（每個人只看得到自己有權限用的功能）。

### 底部「個人偏好」（齒輪圖示）
側邊欄最下面有你的名字，旁邊那顆**齒輪**就是「個人偏好」。點開可以改 4 樣自己的東西：

1. **換頭像** — 上傳一張照片（JPG / PNG）。
2. **換配色** — 4 種主題挑一個：莫蘭迪（米棕，預設）/ 鐵灰 / Airtable 藍 / 克萊因藍。
3. **調字體大小** — 5 段，從極小到特大，看不清楚就調大。
4. **改密碼** — 點「修改密碼」，輸入舊密碼 + 新密碼兩次。

> 改完**馬上生效**，配色和字體大小還會被記住，下次登入維持你選的。

### ⚠️ 容易搞混的一點（教學一定要講）
側邊欄最下面那顆「個人偏好」齒輪，跟選單裡的「**設定**」是**兩個不同東西**：

| | 個人偏好（底部齒輪） | 設定（選單項目） |
|---|---|---|
| 改什麼 | **你自己的**：照片 / 配色 / 字體 / 密碼 | **公司的**：法人資料 / 銀行帳號 / Logo / 獎金政策 |
| 誰能用 | 每個人 | 只有公司管理者 |

一句話記：**齒輪 = 我的偏好；設定 = 公司的設定。**
