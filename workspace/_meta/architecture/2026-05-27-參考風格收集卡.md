## 1. manhattan-editorial — 編輯雜誌風（季刊）

**一句話**：把賣團包裝成「策展季刊」、用編輯排版說品牌故事。

**設計 DNA**
- **季刊敘事框架**：主編的話 → 封面故事 → 4 個精選旅程卡 → manifesto 理念宣言
- **色**：米白／奶油／淺灰打底、accent 極克制
- **字**：serif + sans 混搭、留白大、層級清楚、中英無縫
- **內容呈現**：情緒標語先行（「值得一生的南半球荒野」）、人數／梯次低調列、CTA「閱讀故事」
- **manifesto**：用數字化營運哲學建品牌信任（24 場次／14 人上限）

**值得借的點**
- ✅ 「主編的話 + manifesto」這套**品牌信任建立動線**——官網首頁可直接用
- ✅ 攝影主導 + 文字克制的呼吸感
- ✅ 把「限量／策展」當賣點（少即是貴）

**弱點 / 要注意**
- ⚠️ 純情緒框架、**單團行程細節（每日行程／報價／領隊）塞不進去**——只適合官網品牌層、不適合展示行程詳情頁
- ⚠️ 賣團轉換弱（藏資訊）

## 2. A Minimal — 極簡品牌風

**一句話**：最美、品牌感最強、但賣團力弱。

**設計 DNA**
- 中性色 + 高品質攝影（京都／里斯本／台南 Unsplash）讓目的地自己說話
- tagline「藏在城市轉角的深度旅程」一上來就定品牌調
- 旅程卡：圖為主 + 地點／天數／價格 + 詩意短描述
- 強調體驗細節（「午後三點的市場」「特定茶屋」）勝過制式行程
- 社會證明「8,800+ Travelers Served」

**值得借的點**
- ✅ 品牌調性與第一眼質感最高
- ✅ 詩意短文案模式

**弱點**
- ⚠️ 互動少、搜尋／篩選陽春、行程預覽只到卡片層、賣團力弱 → **當品牌頁、不當主轉換頁**

---

## 3. B Explorer — 探索平衡風

**一句話**：質感 + 資訊最平衡、最貼漫途定位。

**設計 DNA**
- 當代極簡 + 編輯攝影主導、上質感探索品牌
- 中性自然色、accent 克制
- 旅程卡：目的地攝影 + 天數 + 地點 + 評分（4.8–5.0）+ 價格、標體驗標籤（健行／在地美食／攝影）
- 真實旅客推薦（帶照片、非擺拍）
- 規模數字「128+ 路線」「42+ 在地嚮導」、真實「熱門搜尋」
- 強調在地民宿 vs 連鎖飯店

**值得借的點**
- ✅ **質感 + 實用資訊最平衡**——既好看又看得到評分／天數／價格
- ✅ 體驗標籤系統（健行／美食／攝影）= 結構化亮點的雛形
- ✅ 真實旅客證言模式

**弱點**
- ⚠️ 重圖可能拖載入、想快查「日期／可訂」的人視覺層級稍弱

---

## 4. C Elegant — 優雅轉換風

**一句話**：商業完整度最高、轉換導向。

**設計 DNA**
- 現代極簡 + aspirational 攝影、平衡嚮往感與策展真實
- 經典 hero → 搜尋 → 目的地卡 → 證言 區塊動線
- 卡片：目的地照 + 短描述 + 評分 + 價格
- 社會證明強：評論數（★★★★★ 4.9 · 312 reviews）+ 帶肖像的旅客證言
- 在地嚮導檔案（帶評分）、季節促銷 banner

**值得借的點**
- ✅ **轉換導向最完整**——搜尋／評分／證言／促銷 banner 齊
- ✅ 嚮導檔案 + 評分（信任）
- ✅ 資訊層級清楚、好瀏覽

**弱點**
- ⚠️ 長頁資訊較密、雙語偶爾打斷視覺流、卡片分類差異化弱

## Robin 綜合觀察（供寫規範書時參考、未拍板）

1. **三層 token、不是三選一**：A/B/C + manhattan 其實是「**同一組積木 + 不同風格 token**」、可全納進框架當數套起手主題。客戶換 token 就換臉。
2. **官網層 vs 展示行程層的分工**：
   - **官網品牌頁** → manhattan-editorial / A Minimal 的季刊敘事 + 留白調性
   - **展示行程／賣團頁** → B Explorer（平衡）/ C Elegant（轉換）的卡片 + 評分 + 結構化資訊
3. **結構化亮點的雛形已出現**：B 的「體驗標籤」(健行/美食/攝影) + C 的「評分/嚮導」就是上一場談的「讀結構化欄位、AI 排序潤色、業務釘選」三層漏斗的視覺落點。
4. 全部共通：**攝影主導 + 中性色 + 留白 + 中英雙語 + 真實社會證明**——這就是漫途的視覺底層 DNA、可寫進規範書當「不可違反的基調」。
