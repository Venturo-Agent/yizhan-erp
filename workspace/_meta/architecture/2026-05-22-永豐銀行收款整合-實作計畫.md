5. ✅ Samsung Pay（Token Pay API）

### 兩種觸發情境

| 情境 | 觸發點 | 必填 | 流程 |
|---|---|---|---|
| **A. 手動發起收款** | `/finance/payments` 收款單建單頁 | 金額 + Email + 收款方式 | 存檔同時產生付款連結、收款單預存（狀態 = pending）、連結手動或 Email 送客戶 |
| **B. 帳單系統整合** | 客戶自助付款頁 `/pay/[token]`（已存在） | 由客戶勾選想付的 invoice | 客戶可多選帳單合併、按下「我要付款」→ 系統產生對應金額的連結、預存收款單、跳轉刷卡頁 |

### 狀態流（重要）

```
收款單建立瞬間
   ↓
[pending] 待付款／未付款  ← 客戶還沒刷卡
   ↓
[pending_verify] 等永豐回傳  ← 已送出永豐、等 webhook（可選、若 webhook 即時可跳過）
   ↓
[confirmed] 已付款  ← 永豐 webhook 通知刷卡成功
   ↓
[refunded] 已退款（沖正交易、不可逆改）
```

> ⚠️ 紅線 D（不准開作弊後門）：confirmed 後不可塗改、只能開沖正單。

## 3. Phase 1 詳細：明天上線範圍

### Schema 變更（migration 1 份）

```sql
-- 1. payment_methods 加 provider 欄位
ALTER TABLE public.payment_methods
  ADD COLUMN provider VARCHAR(50) DEFAULT 'manual' NOT NULL;

-- 2. 平台層金流商註冊
CREATE TABLE public.platform_payment_providers (
  code VARCHAR(50) PRIMARY KEY,
  provider_name VARCHAR(100) NOT NULL,
  provider_kind VARCHAR(50) NOT NULL,  -- card / wire_transfer / wallet
  enabled BOOLEAN DEFAULT true,
  description TEXT,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);

-- 3. 收款交易紀錄
CREATE TABLE public.payment_transactions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  workspace_id UUID NOT NULL REFERENCES workspaces(id),
  receipt_id UUID REFERENCES receipts(id),
  provider VARCHAR(50) NOT NULL REFERENCES platform_payment_providers(code),

  -- 給客戶的連結（mock 或實際永豐）
  payment_link TEXT,
  payment_link_expires_at TIMESTAMPTZ,

  -- 永豐回傳
  external_order_no VARCHAR(100),  -- oid
  external_trans_no VARCHAR(100),  -- TransNo
  external_approve_code VARCHAR(50),

  status VARCHAR(20) DEFAULT 'pending',  -- pending / authorized / captured / refunded / failed
  raw_webhook_payload JSONB,

  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now(),
  created_by UUID REFERENCES employees(id) ON DELETE SET NULL
);

-- 4. RLS（走 setup_workspace_scoped_rls）
SELECT setup_workspace_scoped_rls('payment_transactions');

-- 5. Seed providers
INSERT INTO platform_payment_providers (code, provider_name, provider_kind, description) VALUES
  ('manual', '手動處理', 'manual', '不接 API、自己處理'),
  ('sinopac_card', '永豐線上刷卡', 'card', 'EPOS URL 付款頁、TransMode=0 一般刷卡'),
  ('sinopac_collect', '永豐豐收款', 'wire_transfer', '虛擬帳號匯款、ATM/網銀'),
  ('sinopac_apple_pay', 'Apple Pay', 'wallet', 'Token Pay API'),
  ('sinopac_google_pay', 'Google Pay', 'wallet', 'Token Pay API'),
  ('sinopac_samsung_pay', 'Samsung Pay', 'wallet', 'Token Pay API');
```
