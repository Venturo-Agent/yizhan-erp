# 如何月結

## 你想做什麼

月底要把這個月所有收款 / 請款 / 出帳 全部結清、產生月損益表 / 試算表、給老闆 / 會計師看。

## 操作步驟

1. **登入 ERP** → 進左側 sidebar
2. **點「會計」** → 「期間結帳」
3. **確認當月所有交易已入帳**：
   - 所有收款都核准了？
   - 所有請款都付款完成 → 產傳票了？
   - 所有結團都跑完？
4. **按「執行月結」**
5. **系統會做**：
   - 鎖定當月所有傳票（不能再改）
   - 產損益表 / 試算表 / 資產負債表
   - 對下個月開新期間

## 重要規矩

⚠️ **月結後不能改舊資料**

- 發現上個月錯了 → 在本月開「沖正交易」（負數抵錯、正數補對）
- 系統設計就是不允許回頭改、會計準則要求

## 報表看哪

月結後到「會計 → 報表」看：

- **損益表**：這月賺多少
- **試算表**：分錄借貸是否平衡
- **資產負債表**：公司財務體質

## 常見問題

**Q：月結後發現有筆收款漏記？**
A：在本月（新期間）補記、會自動分錄到當月。不要去解鎖上個月。

**Q：月結誰能做？**
A：要有「accounting.period_close」capability 的員工才能執行、通常是主管 / 會計。

**Q：月結錯了能 undo 嗎？**
A：不能、月結是不可逆動作。錯了走沖正交易處理。
