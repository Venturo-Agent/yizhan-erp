# 如何新增客戶

## 你想做什麼

把新客戶資料建檔、之後好聯絡 + 串到訂單 + 出團名單。

## 操作步驟

1. **登入 ERP** → 進左側 sidebar
2. **點「客戶管理」**
3. **按右上「新增客戶」**
4. **填寫客戶資訊**：
   - **姓名**（必填）
   - **電話 / Email**
   - **身分證 / 護照號碼**（出團要、可後補）
   - **生日 / 國籍**
   - **聯絡偏好**（LINE / 電話 / Email）
   - **業務員**（指派負責業務、預設為自己）
5. **儲存** → 系統自動產生客戶編號

## 之後可以做什麼

- 在訂單裡指定這個客戶為「下單人」
- 在團員名單中加入這個客戶
- 在 LINE@ 加好友後、系統會自動把 LINE 對話 link 到該客戶

## 常見問題

**Q：客戶資料可以匯入嗎（譬如 Excel）？**
A：目前沒有 bulk import UI、可以聯繫漫途團隊協助。

**Q：客戶刪掉的話會怎樣？**
A：軟刪除（資料還在 DB、只是看不到）、相關訂單 / 出團紀錄不會壞。

**Q：客戶資料只有自己看得到還是全公司？**
A：依公司設定。預設業務員只看自己負責的、主管 / 老闆看全部。
