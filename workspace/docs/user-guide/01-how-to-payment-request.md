# 如何提交請款單（payment_request）

## 你想做什麼

公司付供應商錢、付員工代墊款、付各種開銷 — 都走「請款單」。

## 操作步驟

1. **登入 ERP** → 進左側 sidebar
2. **點「財務」分類** → 「請款管理」
3. **按右上「新增請款單」**（綠色按鈕）
4. **填寫請款資訊**：
   - **付款對象**：供應商 / 員工（下拉選）
   - **請款類別**：交通 / 餐飲 / 住宿 / 雜支 / 薪資 等（下拉選）
   - **金額**：實際付的金額
   - **付款方式**：現金 / 匯款 / 信用卡
   - **掛在哪團**（若有）：選對應旅遊團
5. **儲存** → 系統會自動產生請款單號（譬如 PR2026050001）

## 之後流程

- 主管在「請款管理」核准請款
- 核准後進入「出納」流程
- 出納人員實際付款後、按「已付款」、系統自動產傳票

## 常見問題

**Q：找不到「請款類別」？**
A：到「財務 → 設定 → 團體請款類別 / 公司支出項目 / 公司收入項目」自己加新類別。

**Q：付款對象沒有我要的供應商？**
A：先到「供應商管理」新增該供應商、再回來請款。

**Q：請款後可以改嗎？**
A：未核准前可改、已核准 / 已付款後不能改、要走「沖正」處理。
