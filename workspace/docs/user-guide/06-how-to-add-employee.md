# 如何新增員工

## 你想做什麼

公司來了新人、要建檔、給帳號、設權限、好開始用 ERP。

## 操作步驟

1. **登入 ERP** → 進左側 sidebar
2. **點「人資 HR」** → 「員工管理」
3. **按右上「新增員工」**
4. **填寫員工資訊**：
   - **姓名 / 英文名**
   - **Email**（用來登入）
   - **電話 / 身分證 / 出生日期**
   - **部門 / 職位**（從下拉選）
   - **角色**（業務 / 會計 / HR / 主管 等）
   - **薪資資訊**（月薪 / 時薪 / 試用期）
5. **儲存** → 系統自動產生員工編號（譬如 E001）

## 給帳號

- 系統會自動發 invite email 給該員工
- 員工收到後、設密碼登入
- 第一次登入會看到自己權限對應的 sidebar 功能

## 權限設定

- **角色（role）** 決定看到哪些功能（業務只看客戶 / 訂單、會計看財務）
- 細節到「**人資 → 角色權限**」自定義每個 role 能做什麼
- 預設角色：業務 / 會計 / HR / 主管 / 老闆 / 系統管理員

## 常見問題

**Q：員工離職怎處理？**
A：到員工頁按「離職」、設定離職日。系統會自動關閉帳號、但歷史紀錄不會刪。

**Q：員工權限不夠 / 太大怎調？**
A：改 role 或調對應 role 的 capability。

**Q：可以一次匯入多個員工嗎？**
A：目前沒有 bulk import UI、聯繫漫途團隊協助。
