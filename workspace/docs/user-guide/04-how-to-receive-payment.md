# 如何收款

## 你想做什麼

客戶付團費了、要把這筆錢記到系統、好對應訂單 + 後續結團。

## 操作步驟

1. **登入 ERP** → 進左側 sidebar
2. **點「財務」** → 「收款管理」
3. **按右上「新增收款」**
4. **填寫收款資訊**：
   - **客戶**：選下單客戶
   - **訂單 / 團號**：對應哪張訂單 / 哪團
   - **金額**：客戶實付金額
   - **收款方式**：現金 / 匯款 / 信用卡 / LINE Pay 等
   - **收款日期**
   - **備註**（譬如「分期 1/3 期」）
5. **儲存** → 系統自動產生收款單號

## 主管核准

- 收款預設「待核准」
- 主管 / 會計到「收款管理」核准
- 核准後系統自動產生傳票（入帳）

## 收費 vs 已收款

- 訂單頁可看「應收金額 vs 已收金額」對照
- 已收 < 應收 = 還有尾款沒收
- 系統會自動提醒未收款的團

## 常見問題

**Q：客戶分期付款怎處理？**
A：每期分開記一筆收款、備註寫「1/3 期」「2/3 期」、系統自動加總。

**Q：客戶退款怎麼做？**
A：到「請款管理」開「退款請款」、走出納流程退錢給客戶。

**Q：手續費（譬如刷卡 3%）怎處理？**
A：核准收款後、系統會自動產生「手續費請款單」、會計確認後付給銀行。
