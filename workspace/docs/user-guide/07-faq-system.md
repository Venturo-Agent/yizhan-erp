# 一棧 ERP 常見問題

## 帳號 / 登入

**Q：忘記密碼怎辦？**
A：登入頁點「忘記密碼」、輸 Email、收信點 reset link。

**Q：第一次登入看不到任何功能？**
A：沒設權限。請主管 / HR 去「人資 → 角色權限」開對應 capability。

**Q：怎麼切換不同公司（workspace）？**
A：右上角頭像點下拉、選「切換 workspace」。只有有權限的 workspace 看得到。

## 功能 / 操作

**Q：找不到「XX 功能」怎辦？**
A：
1. 看 sidebar 主分類（旅遊團 / 客戶 / 財務 / 會計 / HR 等）展開找
2. 或用 sidebar 上方搜尋
3. 還是找不到 → 可能是你公司沒買這個 module、或 HR 沒給你權限

**Q：為什麼看到某些選項是灰的 / 點不下去？**
A：通常代表你沒權限。聯繫主管 / HR 加 capability。

**Q：系統有支援 mobile app 嗎？**
A：目前 web 版優先、mobile app 在開發中（Phase 2）。

## 客戶 / 訂單 / 團

**Q：客戶資料只有我看得到還是全公司？**
A：預設業務只看自己負責的、主管 / 老闆看全部。
HR 可以調整。

**Q：團員是不是要客戶都建檔才能加？**
A：對、先建客戶資料、才能加進團員列表。

## 財務 / 會計

**Q：請款 vs 出納差別？**
A：請款 = 申請付這筆錢（譬如業務代墊餐費要報帳）、出納 = 實際付出去（譬如會計開銀行匯款）。

**Q：收款後可以改嗎？**
A：未核准前可改、已核准後不能改、要走「沖正」處理。

**Q：月結後發現錯怎辦？**
A：在當月開沖正交易、不要去解鎖上個月。

## 系統 / 整合

**Q：可以串 LINE 客戶嗎？**
A：可以。「AI Hub → 設定 → LINE」綁定貴公司 LINE@、客戶來訊自動進「對話收件夾」。

**Q：可以串 Facebook / Instagram 嗎？**
A：可以。同上、在 AI Hub 設定 Facebook / Instagram channel。

**Q：可以匯出資料嗎？**
A：各個列表頁右上有「匯出 Excel」按鈕。報表頁可以匯出 PDF。
