# 如何做成本轉移 / 收款轉移

## 你想做什麼

A 團的請款 / 收款記錯團了、要轉到 B 團；或客戶從 A 團換到 B 團、要把已收款轉移過去。

「轉移」= 不刪原紀錄、不開新單、系統用「對沖」方式雙方做帳：
- 從 A 團出沖正、寫到 B 團
- 兩邊都有 audit trail、會計師看得懂

## 成本轉移（請款已開、要移到別團）

1. **登入 ERP** → 進「財務管理 → 請款管理」
2. **點該筆請款單**（已開出的）→ 打開請款 dialog
3. **dialog 內按「成本轉移」按鈕**
4. **選目標團**（從下拉清單）
5. **勾要轉移的項目**（可整筆 / 部分品項）
6. **確認** → 系統自動：
   - 在原團開沖正請款（負金額抵銷）
   - 在目標團新開請款（正金額補上）
   - 兩筆用 `transferred_pair_id` 關聯、會計報表自動追蹤

## 收款轉移（收款已 confirmed、要移到別團）

1. **登入 ERP** → 進「財務管理 → 收款管理」
2. **點該筆收款**（必須是「已 confirmed」狀態）→ 打開收款 dialog
3. **dialog 內按「收款轉移」按鈕**
4. **選目標團 + 轉移金額**
5. **確認** → 同樣對沖式做帳：
   - 原團負金額沖銷
   - 目標團正金額補上

## 常見問題

**Q：什麼時候用成本轉移、什麼時候用收款轉移？**
A：
- 成本轉移：請款（你付出去的錢）算錯團、譬如機票本來掛 A 團、應該掛 B 團
- 收款轉移：收款（客戶付給你的錢）算錯團、譬如客戶從 A 團換 B 團

**Q：轉移後可以撤銷嗎？**
A：能。再做一次反向轉移即可。

**Q：找不到「轉移」按鈕？**
A：兩個都藏在 dialog 內、不在列表頁。
- 成本轉移：打開請款單編輯 dialog
- 收款轉移：打開收款編輯 dialog（且收款必須已 confirmed）

**Q：為什麼不能改原單就好、要用轉移？**
A：會計準則要求 audit trail。改原單看不出歷史、轉移留兩邊紀錄、稅務 / 審計能追溯。
