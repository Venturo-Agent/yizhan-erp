# 如何開新旅遊團

## 你想做什麼

要出一個新團、需要在系統建檔、之後好接訂單 + 出團 + 結團。

## 操作步驟

1. **登入 ERP** → 進左側 sidebar
2. **點「旅遊團」** → 看到團體列表
3. **按右上「新增旅遊團」**
4. **填寫團體資訊**：
   - **團名**（譬如「2026 春節東京賞櫻 5 日」）
   - **出發日期 / 回程日期**
   - **目的地 / 國家 / 地區**
   - **航空公司 / 航班資訊**
   - **業務員**（負責業務、預設為自己）
   - **領隊 / 導遊**（可後補）
   - **團型**：散客拼團 / 包團 / 自由行
   - **預計人數**（最低 / 最高）
5. **儲存** → 系統自動產生團號（譬如 T2026050001）

## 之後流程

- 客戶下訂單時、選這個團
- 訂單轉成「團員」列表
- 出發前、印「出團名單」「行程表」「保險清單」
- 回程後、執行「結團」、會計開始入帳

## 常見問題

**Q：團還沒收滿能開嗎？**
A：能、預設「招生中」狀態。報名人數累積、自由 monitor。

**Q：團如何取消？**
A：在團體頁面按「取消」、系統會問你「已下訂客戶的退款怎處理」、引導你做退款請款。

**Q：拼團跟包團差別？**
A：散客拼團 = 多客戶湊一團、包團 = 一個客戶（或公司）整團包下。
