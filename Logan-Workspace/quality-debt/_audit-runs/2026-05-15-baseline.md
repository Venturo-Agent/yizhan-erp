## Loop 啟動建議

短期（每天）：
- 每天早上跑 `npm run audit:rls + audit:capability-coverage + audit:flow-strictness`
- 結果寫進 `_audit-runs/{date}.md`

中期（一週後）：
- CronCreate 半夜 3 點自動跑、寫 diff 報告
- 退步 → telegram 通知 + 開 issue

長期（一月後、開會討論）：
- 半夜 claude loop 自動 fix 小違規（缺 audit context / 缺 capability check）
- 大違規寫 issue + 通知
